## Задача "Сумма цифр числа"

Дано число, необходимо вывести сумму цифр этого числа.

**Входные данные**

>Программа получает на вход число.

**Выходные данные**

>Программа должна вывести сумму цифр данного числа.

**Примеры**

**Входные данные**
```
1111
```
**Выходные данные**
```
4
```
---
**Входные данные**
```
1234
```
**Выходные данные**
```
10
```
